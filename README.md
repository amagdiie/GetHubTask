# GetHubTask
Git hub task to login with git hub and return user data and also get all repo using Koin and Ktor and Pagination and Compose Ui and Kotlin and MVVM

## Overview
This is a small, production-quality Android application that authenticates with GitHub, displays the authenticated user's repositories, and lists branches for a selected repository. The app is built with a focus on clean architecture, MVVM, testability, and engineering best practices.

**Key Features:**
- GitHub OAuth authentication
- Paginated list of repositories with search/filter functionality (paginated)
- Sign out functionality, clearing sensitive data
- Optional UX enhancements: pull-to-refresh, repository language chip, branch protection indicators, deep link support

**Tech Stack:**
- Kotlin 2.0.0+
- Android Gradle Plugin 8.6.1+
- Jetpack Compose UI
- Target SDK: API Level 35+
- Clean Architecture + MVVM
- Networking: Ktor
- Local caching and offline support
- Unit & UI testing
